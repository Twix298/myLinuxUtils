<h2>Утилита myWc</h2> 
Подсчет слов в фале
~$ ./myWc -w input.txt
777 input.txt
# Подсчет линий в файлах input2.txt and input3.txt
~$ ./myWc -l input2.txt input3.txt
42 input2.txt
53 input3.txt
# Подсчет символов в файлах input4.txt, input5.txt and input6.txt
~$ ./myWc -m input4.txt input5.txt input6.txt
1337 input4.txt
2664 input5.txt
3991 input6.txt
